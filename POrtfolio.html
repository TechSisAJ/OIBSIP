Project 1

Imototo was the very first website I created as my Capstone project during my days as a student in Ekiti State Digital Skilling Academy. It was created to solve the problem of waste management in Ekiti State.

Priotization Framework Used
Home Page: The home page introduces the user to what the page is all about as seen in the picture above.
Our Services Page: This page tells the user the services rendered by my client such as refuse pick up using the map, Dumpster rental, recycling, pricing and subscriptions using paystack API.
Service Area page: This page shows the area my clients services can be rendered as the service is limited to just Ekiti state residents.
Contact Page:This page is for my client's potential customer to contact them incase of any challenges while using the page or for feedbacks.
